# Roupix

Roupix is a community crypto money. It target community of any size to reward exchange of services between the member of a given community.

It has the following characteristics:

- Each member of the community starts with a given amount of money.
- The money is melting which make stupid keeping a big amount of money and push member to use the money.
- Every weeks, to compensate the melting money, money is created equaly on each member account.
- By democratic consensus, the money can crash and be reset: every member will recover it's initial.
